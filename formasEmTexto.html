Shape-outside

Essa propriedade permite que voc~e desenhe uma forma para que o texto se envolva e flua ao seu redor

junto a essas propriedades temos formas básicas

O elemento que vamos aplicar a forma com propriedade Shape-outside

Float
  • inset()
  • circle()
  • ellipse()
  • polygon()

inset()

POde ser usado para criar um retangulo ou quadrado com um deslocamento opcional para o texto que o envolver
